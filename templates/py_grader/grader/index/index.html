About

I developed this tool on my own after being a teacher's asistant for a while and getting tired of manually grading code for some classes. It uses Django and Bootstrap and it's my first web project.

PyGrader

PyGrader Instructor Home

A system for automatically grading Python code used by CLASS/INSTITUTION HERE

Create Assignment

Create an Assignment using a key file and test cases. Assignments have an open and close time and a limited number of submissions.

Test Submit Assignment

Submit a Python file and view the grade it recieves without recording it.

View Assignments

View created assignments and data about submissions. View and export grades.

View Submission

View a submission result with a submission ID and see which test cases pass and the resulting grade as well as the NetID, name and IP address of the student.

Manage NetIDs

Add, remove, and upload NetIDs which allow students to submit assignments.
